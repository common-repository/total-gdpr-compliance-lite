<strong>Total GDPR Compliance Lite </strong> is the plugin for you to help make your site GDPR compatible with single plugin with it’s essential but very easy configurations. 
This plugin is meant to assist you automating the multiple steps required, compiling various services into single place incorporating the GDPR laws and regulations, under the European Union. 
While we can't ensure the absolute or 100% compliance to GDPR laws, this plugin is to make you one step closure to meeting the obligations, that falls under the GDPR regulation. Please do contact your legal advisory to ensure that all requirements are met.

== Installation ==
1. Unzip total-gdpr-compliance-lite.zip
2. Upload all the files to the /wp-content/plugins/total-gdpr-compliance-lite
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on Total GDPR Compliance Lite on left hand side of admin menu.

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides the ability to allow site owner to make the site gdpr compatible.


= Will it be difficult to design cookie layout? =
You won't need to get any trouble regarding design and layout of the cookie or any other settings since we have compatible and responsive designs.You can choose any as per your requirement or use custom designs too to match your site requirement.


= Where can I find the documentation for the plugin? =
Complete documentation can be found  [here](https://accesspressthemes.com/documentation/total-gdpr-compliance-lite/).


= Can I use Cookie Template wherever I want? =
Yes. You can just enable cookie on <strong>Cookie Setting</strong> page and select where you want to display cookie info.
